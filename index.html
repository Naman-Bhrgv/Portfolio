Research

    Sentiment Analysis of the United States Public Support of Nuclear Power on Social Media Using Large Language Models

    I designed and implemented a Data Mining pipeline to interact with X(formely Twitter) API, that was pivotal in extracting tweets concerning Nuclear Engineering. Further, I analyzed over one million tweets using BART model to extract main themes that arose from these online discussions. To detect the sentiment of tweets, I fine-tuned multiple machine learning and deep learning models, with leading to an accuracy of .

    Efficient selection of SAR features using ML based algorithms for accurate FVC estimation

    I constructed an XGBoost model with an RMSE of 0.159 for the purpose of estimating the Fractional Vegetation Cover (FVC) parameter, utilizing SAR data for wheat and sugarcane crops. In order to enhance the efficiency of the model, I employed feature selection techniques to identify the five most critical features for accurate prediction. This contributed to a significant reduction in prediction time across agricultural fields, streamlining the process for deploying model for prediction over large agricultural fields.

    A Design Pattern based Forecasting Model for Predicting Time Series data in Kaggle Ecosystem

    I developed a SARIMA (Seasonal AutoRegressive Integrated Moving Average) model in Python, which achieved a mean absolute percentage error (MAPE) of 11%. To enhance the system, I optimized the data extraction and prediction process using design patterns, specifically the Factory and Adapter Patterns, which not only improved the code maintainability by 46.18% but also substantially reduced its complexity by 73.5%.

    Machine learning models for prediction of xenobiotic chemicals with high propensity to transfer into human milk

    I created a comprehensive dataset of 375 chemicals, by computing and aggregating their physical and chemical properties from different softwares. Further, I developed a Support Vector Machine (SVM) model to assess the contamination of maternal milk from environmental chemicals. The model, delivering an accuracy of 77.33% and an impressive specificity of 84%.

Projects

Crop Anomaly Detection using Time Series Forecasting

Developed a time series forecasting model to detect anomaly during early stage of crop growth. Extracted multiple statistical and temporal features from time series using python libraries. Further, I trained Generative Adversarial Networks to balance the data distribution. Xgboost model was trained on the balanced data to predict crop anomaly.
